# Public Neovim Config

## This config was designed on an arch linux only system and results may vary depending on the distro

### Prerequisites packages

List of packages that are needed/HIGHLY recommended
* ripgrep
* gcc 
* npm
* xclip

### Command for copying this repo to you config 
```bash
git clone https://github.com/Boofny/PubNvim.git ~/.config/nvim
```

### Remove git folder to make you own conf repo
```bash
rm -rf ~/.config/nvim/.git
```

### Now run noevim with command:
```bash
nvim
```
